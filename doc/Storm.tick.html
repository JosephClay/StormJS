Namespace: tick

Storm. tick

Hook into requestAnimationFrame through Storm. Keeps a single raf so that there aren't multiple calls or miss-calls that would cause raf from functioning, e.g.: var update = function() { raf(update); }; raf(update); later in the application: raf(update); Also automatically calls TWEEN if it's present https://github.com/sole/tween.js/
